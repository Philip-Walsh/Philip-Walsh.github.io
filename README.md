## How to Run Locally

1. Clone the repository: `git clone https://github.com/Philip-Walsh/Philip-Walsh.github.io.git`
2. Navigate to the project directory: `cd Philip-Walsh.github.io`
3. Install dependencies: `npm install`
4. Run locally: `npm start`
